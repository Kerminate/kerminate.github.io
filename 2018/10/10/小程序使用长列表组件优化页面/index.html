长列表的使用方法

安装长列表

  • 使用 npm 安装
    npm install --save miniprogram-recycle-view
    在页面的 json 文件加入 recycle-view 和 recycle-item 自定义组件的配置

由于小程序从 2.2.1 版本开始才支持npm安装第三方包，考虑到目前还有接近 10% 的用户的基础库低于这个版本，所以我更推荐大家使用下面一种方法

  • 将长列表的源码拷贝一份到项目目录下
    json文件里配置就是对应的代码路径

菜单页的目录结构

在 menu.ts 里管理 recycle-view 的数据

因为长列表的特性所以数据必须是一维的，而我们的菜品数据是二维的(菜品下是菜品分类的数组，每个分类下才是具体的菜品信息)
我们需要对后端传来的数据进行处理。

处理长列表的数据
因为长列表是一维数据，里面包括了题目信息和菜品信息，所以在页面加载完成前预留的空间是不一样的，所以在 itemSize 函数里通过每个 item 的属性生成不同的宽高。transformRpx 函数可以将 rpx 转化为 px。
这里主要是题目和菜品两个不同的子项，而菜品因为内部显示信息的不同，高度也会有所变化，展示所有信息时，高度会被撑开。所以长列表里共有三种不同高宽的子项。
